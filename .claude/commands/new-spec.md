# /new-spec

Start a new Feature Spec.

1. Ask for the feature folder name: `{TICKET-ID}-{slug}` (e.g. `Story-0002-user-registration`).
2. Create the folder under `plan/spec/`.
3. Load `plan/reference/glossary.md`, `personas.md`, `roles.md`.
4. Help draft `FS-XXX.md` — propose ACs one at a time, cross-check against `contracts/`.
5. Present the draft and STOP. The human owns the final FS.
