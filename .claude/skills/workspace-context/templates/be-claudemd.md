## Key Patterns

- **App factory**: `create_app()` in `src/main.py` — all middleware, routes, and metrics registered here
- **Tenant isolation**: `X-Tenant-ID` header required on all API requests
- **Request ID propagation**: `X-Request-ID` header echoed in responses
- **Error responses**: Consistent `{"code": "...", "message": "..."}` format
- **Test naming**: `test_ts_{feature}_{scenario}_{description}` maps to TS scenarios
